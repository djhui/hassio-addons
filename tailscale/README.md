## 关于

Tailscale 是一个零配置的 VPN，几分钟内可以在任何设备上安装，包括您的 Home Assistant 实例。

在您的服务器、计算机和云实例之间创建一个安全网络。即使被防火墙或子网隔离，Tailscale 也能正常工作。Tailscale 为您管理防火墙规则，并在您所在的任何地方都能工作。
