针对usenet和torrent用户的电影组织/管理工具。

## 关于

[Radarr] 是一个用于Usenet和BitTorrent用户的电影收藏管理器。它可以监控多个RSS源获取新电影，并将与客户端和索引器接口以获取、排序和重命名它们。它还可以配置为在图书馆中的现有文件有更高质量格式可用时，自动升级这些文件的质量。
