## 关于

[Radarr] 是一个针对Usenet和BitTorrent用户的电影收藏管理器。它可以监控多个RSS源以获取新电影，并会与客户端和索引器接口，抓取、排序和重命名它们。它还可以配置为在更好的质量格式可用时，自动升级库中现有文件的质量。
