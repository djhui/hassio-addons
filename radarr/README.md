## 关于

[Radarr] 是针对Usenet和BitTorrent用户的电影收藏管理器。它可以监控多个RSS源以获取新电影，并将与客户端和索引器对接，以抓取、排序和重命名这些电影。它还可以配置为在库中存在的文件有更好的质量格式可用时，自动升级这些文件的质量。
