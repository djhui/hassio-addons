# VNC Viewer

将 noVNC 应用程序捆绑在 Home Assistant 插件中。 这允许将任何 VNC 服务器集成到 Home Assistant 中。

![Addon Stage][stage-badge]

## 注意

目前只支持一个连接，如果需要更多连接，请在仓库讨论中创建或加入线程。
