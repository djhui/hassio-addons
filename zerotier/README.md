## 关于

ZeroTier 通过一个系统提供了 VPN、SDN 和 SD-WAN 的功能。像管理整个世界是一个单一数据中心一样，管理您在本地和广域网中的所有连接资源。

人们使用 ZeroTier 无缝连接笔记本电脑、台式机、手机、嵌入式设备、云资源以及随处都能使用的应用程序。它将整个世界转变为一个单一的数据中心，您现在可以通过此插件将您的 Home Assistant 实例添加到其中。
