## 关于

Tautulli 是一款可以与 Plex 媒体服务器同时运行的应用程序，
用于监控活动并跟踪各种统计数据。

最重要的是，这些统计数据包括观看过的内容、
是谁观看的、何时何地观看的以及如何观看的。

所有统计数据都以漂亮且干净的界面呈现，包含许多表格和
图表，这使得向其他人炫耀你的服务器变得很容易。
