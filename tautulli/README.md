## 介绍

Tautulli 是一款可以与你的 Plex 媒体服务器一起运行的应用程序，用于监控活动和跟踪各种统计数据。

最重要的是，这些统计数据包括了观看了什么，谁观看了，何时何地观看的，以及如何观看的。

所有统计数据都以美观简洁的界面呈现，包含许多表格和图表，使你可以轻松向其他人炫耀你的服务器。
