# OWASP Juice Shop

[OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) 可能是现代最复杂和最先进的不安全web应用程序！它可以用于安全培训、意识演示、CTF以及作为安全工具的实验品！Juice Shop 包含了整个 OWASP 十大漏洞以及许多其他在现实应用程序中发现的安全缺陷！
