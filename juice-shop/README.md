# OWASP Juice Shop

[OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) 可能是最现代和最复杂的不安全Web应用程序！它可以用于安全培训、意识演示、CTF以及作为安全工具的实验对象！Juice Shop 包含整个 OWASP 十大中的漏洞以及在实际应用中发现的许多其他安全缺陷！
