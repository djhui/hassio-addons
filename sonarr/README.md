## 关于

[Sonarr] 是一个适用于 Usenet 和 BitTorrent 用户的 PVR。它可以监控多个 RSS
源以获取您最喜欢的节目的新剧集，并会抓取、排序和重命名它们。它还可以配置为在更好的质量格式可用时自动升级已经下载的文件质量。
