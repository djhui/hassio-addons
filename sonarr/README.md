适用于新闻组和 bittorrent 用户的智能 PVR。

## 关于

[Sonarr] 是一款针对 Usenet 和 BitTorrent 用户的 PVR。它可以监控多个 RSS 源，以获取您最喜欢节目的新剧集，并会自动抓取、分类和重命名这些剧集。当有更高质量的文件格式可用时，它还可以配置为自动升级已经下载的文件的质量。
