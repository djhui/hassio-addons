针对新闻组和比特流用户的智能PVR。

## 关于

[Sonarr] 是一款针对Usenet和BitTorrent用户的PVR。它可以监控您最喜欢的节目的多个RSS源，以获取新剧集，并会抓取、排序并重命名它们。它还可以配置为在更高质量的格式可用时，自动升级已经下载的文件的质量。
