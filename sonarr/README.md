针对新闻组和种子用户的智能PVR。

## 关于

[Sonarr]是一个针对Usenet和BitTorrent用户的PVR。它可以监控多个RSS
源，以获取您喜爱的节目的新剧集，并将其抓取、排序和重命名。它还可以配置为在更好的质量格式可用时自动升级已下载文件的质量。
