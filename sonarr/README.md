为新闻组和种子用户提供智能 PVR。

## 关于

[Sonarr] 是一个针对 Usenet 和 BitTorrent 用户的 PVR。它可以监控多个 RSS
源，以获取您喜欢节目最新集数，并将其抓取、分类和重命名。它还可以配置为在更高质量的格式可用时，自动升级已经下载的文件质量。
