## 关于

[Sonarr] 是一个用于 Usenet 和 BitTorrent 用户的 PVR。它可以监控多个 RSS
源，以获取您喜欢的节目的新剧集，并会自动抓取、排序和重命名
它们。它还可以配置为在可用更高质量的格式时，自动升级已下载文件的质量。
