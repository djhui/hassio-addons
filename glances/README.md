Glances是一个用Python编写的跨平台系统监控工具。

## 关于

Glances是一个跨平台的监控工具，旨在通过基于Web的界面在最小的空间内呈现尽可能多的信息。

Glances可以将所有系统统计数据导出到InfluxDB，让您能够查看所有系统信息及其随时间的行为。
