# OWASP ZAP Home Assistant 插件

# 🕸️ 已弃用

世界上最受欢迎的免费网络安全工具，现在作为 Home Assistant 插件。
我个人偶尔使用它来检查物联网设备的数据。

更多信息请访问 [官方网站](https://www.zaproxy.org/)。

## 配置

目前插件本身没有配置选项。设置可以通过 ZAP 自身的用户界面进行调整。

### Raspberry Pi

在 Raspberry Pi 4 上，ZAP 的启动非常缓慢。请注意。
