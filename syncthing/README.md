# Syncthing 插件

[**Syncthing**](https://syncthing.net/) 包裹在 [Home Assistant Supervisor 插件](https://www.home-assistant.io/addons/) 中。

> Syncthing 是一个持续的文件同步程序。它在两个或多个计算机之间实时同步文件，安全地防止窥探。你的数据是你自己的，你有权选择存储位置，该数据是否与第三方共享，以及如何通过互联网传输。
