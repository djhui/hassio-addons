# Syncthing 附加组件

[**Syncthing**](https://syncthing.net/) 封装在 [Home Assistant Supervisor 附加组件](https://www.home-assistant.io/addons/) 中。

> Syncthing 是一个持续的文件同步程序。它在两个或多个计算机之间实时同步文件，安全地保护用户免受窥视。您的数据仅属于您自己，您有权选择数据存储的位置，是否与第三方共享，以及如何通过互联网传输。
