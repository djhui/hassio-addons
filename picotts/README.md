# PicoTTS (Hass.io 插件)

[440/naggety/picotts](https://github.com/naggety/picotts) 作为 Home Assistant Supervisor 插件。 (可用于 Raspberry Pi 3-)

这个插件执行后端的 tts 引擎并提供一个网络服务。它支持所有默认语言。
