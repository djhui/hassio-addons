# PicoTTS (Hass.io 插件)

[ مورد: naggety/picotts](https://github.com/naggety/picotts) 作为 Home Assistant Supervisor 插件。（可在 Raspberry Pi 3- 上使用）

这个插件通过一个网络服务执行 tts 引擎。它支持所有默认语言。
