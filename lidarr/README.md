看起来和闻起来像 Sonarr，但为音乐而制。

## 关于

[Lidarr] 是一个为 Usenet 和 BitTorrent 用户设计的音乐收藏管理器。它可以监控多个 RSS 源，以获取您最喜爱的艺术家的新曲目，并自动抓取、排序和重命名它们。它还可以配置为在更高质量格式可用时自动升级已下载文件的质量。

[Lidarr]: https://lidarr.audio/
