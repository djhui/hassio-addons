看起来和闻起来都像 Sonarr，但它是为音乐制作的。

## 关于

[Lidarr] 是一个为 Usenet 和 BitTorrent 用户设计的音乐收藏管理器。它可以监控来自您最喜爱的艺术家的新曲目的多个 RSS 源，并会自动抓取、分类和重命名这些曲目。当有更高质量的格式可用时，它还可以配置为自动升级已经下载的文件的质量。

[Lidarr]: https://lidarr.audio/
