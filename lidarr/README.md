看起来和闻起来都像 Sonarr，但专为音乐而设计。

## 关于

[Lidarr] 是一个适用于 Usenet 和 BitTorrent 用户的音乐集合管理器。它可以监控多个 RSS 源，以获取您最喜欢艺术家的新曲目，并对其进行抓取、分类和重命名。它还可以配置为在更高质量的格式可用时，自动升级已经下载的文件的质量。

[Lidarr]: https://lidarr.audio/
