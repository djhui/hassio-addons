# Home Assistant 插件：CEC 扫描器

## 安装

按照以下步骤在您的系统上安装插件：

1. 在您的 Home Assistant 前端中导航到 **设置** -> **插件** -> **插件商店**。
2. 找到“CEC 扫描器”插件并点击它。
3. 点击“安装”按钮。

## 如何使用

该插件没有配置，开箱即用。

1. 启动插件。
2. 检查插件日志输出以查看结果。
