图书管理和自动化（电子书的 Sonarr）。

## 关于

[Readarr] 是一个用于 Usenet 和 BitTorrent 用户的电子书和有声书收藏管理器。它可以监控多个 RSS 源以获取你最喜欢的作者的新书，并会抓取、排序并重命名它们。请注意，仅支持每本书的一种类型。
