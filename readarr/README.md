书籍管理器和自动化工具（适用于电子书的 Sonarr）。

## 关于

[Readarr] 是一个为 Usenet 和 BitTorrent 用户设计的电子书和有声书收藏管理器。它可以监视多个 RSS 源，以获取您最喜欢的作者的新书，并会抓取、排序和重命名它们。请注意，仅支持一种特定类型的书籍。
