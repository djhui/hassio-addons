UniFi 网络应用程序允许您使用网页浏览器管理您的 UniFi 网络。

## 关于

此插件运行 Ubiquiti Networks 的 UniFi 网络应用程序软件，允许您通过网页浏览器管理您的 UniFi 网络。该插件提供了一键安装和运行的解决方案，便于 Home Assistant 用户轻松启用、运行和更新他们的网络。

此插件支持所有 Home Assistant 支持的架构，包括 Raspberry Pi。
