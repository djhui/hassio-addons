## 关于

InfluxDB 是一个开源时间序列数据库，优化用于高写入量。
它用于记录指标、传感器数据、事件，并执行分析。它提供一个 HTTP API 供客户端交互，并且
通常与 Grafana 结合使用，以可视化数据。

此附加组件还预装了 Chronograf 和 Kapacitor。这
为您提供了一个漂亮的 InfluxDB 管理界面，用于管理您的用户、数据库、
数据保留设置，并让您通过数据浏览器查看数据库内部。
