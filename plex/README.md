你的录制媒体、直播电视、在线新闻和播客，完美整理并准备好进行流式播放。

## 关于

Plex 插件将你最喜欢的媒体集中在一个地方，使其美观且易于享受。这个插件提供的 Plex 媒体服务器，组织你的个人视频、音乐和照片集合，并将其流式传输到你所有的设备上。
