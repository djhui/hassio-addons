## 关于

Folding@home（FAH或F@h）是一个分布式计算项目，用于执行蛋白质动态的分子动力学模拟。其最初的重点是蛋白质折叠，但已转向更多生物医学问题，例如阿尔茨海默病、癌症、埃博拉和冠状病毒。

该项目利用安装了软件的志愿者个人电脑的空闲处理资源。

现在，您可以将Home Assistant实例的空闲时间捐赠给Folding@home项目，帮助抗击这些疾病。
