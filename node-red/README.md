## 关于

[Node-RED][nodered] 是一个编程工具，用于以新颖有趣的方式将硬件设备、API 和在线服务连接在一起。

它提供了一个基于浏览器的编辑器，使得使用调色板中的各种节点轻松地连接流程，这些节点可以一键部署到其运行时。
