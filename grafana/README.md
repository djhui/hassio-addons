## 关于

适用于您所有指标的分析平台。

Grafana 允许您查询、可视化、警报和了解您的指标，无论它们存储在哪里。创建、探索和分享仪表板。使用炫酷和引人入胜的图表，以及其他数据可视化，了解您的家庭自动化系统。

将此插件与 InfluxDB 插件结合使用，以获得对您家庭的强大洞察。
