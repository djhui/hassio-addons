## 关于

[Grocy][grocy] - 超越你冰箱的 ERP 是一个强大的杂货和家庭管理解决方案，为您的家庭提供如下功能：

- 库存管理
- 购物清单
- 食谱
- 家务与任务
- 库存
- 以及更多。
