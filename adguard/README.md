## 关于

AdGuard Home 是一个网络范围的广告和追踪器屏蔽 DNS 服务器，具有家长控制（成人内容屏蔽）功能。它的目的是让您控制整个网络和所有设备，不需要使用客户端程序。

AdGuard Home 提供一个美观、易于使用且功能丰富的 web 界面，以便轻松管理过滤过程及其设置。
