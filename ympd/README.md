# ympd

[ympd](https://ympd.org/) 是一个轻量级的 MPD（音乐播放器守护进程）网页客户端，它无需专用的网络服务器或像 PHP、NodeJS 或 Ruby 这样的解释器即可运行。它经过调整以最小化资源使用，仅需很少的依赖。
